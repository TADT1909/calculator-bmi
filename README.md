# calculator-bmi

API tính toán chỉ số Sức Khỏe (BMI) dựa theo cân nặng và chiều cao cho #chatfuel

Url: index.php?chiso=cannang_chieucao
Trong đó: cannang_chieucao được cấu thành bằng Cân Nặng _ Chiều Cao
+ Cân nặng tính được tính bằng Ký-lô-gam (kg).
+ Chiều cao tính bằng cen-ti-met (cm) 

VD: http://tuanxuong.com/api/bmi.php?chiso=60_173
=> Nghĩa là: Tính chỉ số BMI dựa trên Cân nặng 60 kg và chiều cao là 173 cm (1 mét 73)

DEMO: https://m.me/1822817251268721
